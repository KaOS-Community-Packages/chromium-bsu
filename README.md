chromium-bsu
=====
A fast paced top scrolling shooter

URL: http://chromium-bsu.sourceforge.net/

Needed from KCP:
[freealut](../../../freealut)

Install in this order:
```
kcp -i freealut --asdeps 
```
```
kcp -i chromium-bsu
```
